## 📌 项目简介
本项目要求开发一个 **基于 Java GUI 的汽车租赁管理系统**，包含车辆管理、租借管理、用户管理、报表与数据持久化功能。  
占 **课程作业 30%**。

## 👥 用户角色与功能

### 🔑 管理员（Admin）
- 用户管理（增删改查）
- 车辆管理（品牌、型号、年份、车牌、颜色、状态）
- 查看报表（租借记录、车辆可用性、收入）
- 系统日志查看

### 👤 员工（Staff Member）
- 办理租借与归还（记录开始与结束日期）
- 更新车辆状态
- 搜索车辆（品牌 / 型号 / 车牌）
- 管理顾客信息与历史记录

## 🗄️ 数据管理
- 必须使用 **关系型数据库**（MySQL / PostgreSQL / SQLite）  
- 数据包含：车辆、用户、租借记录  

若无法使用数据库，可写明原因并使用文件系统替代。
