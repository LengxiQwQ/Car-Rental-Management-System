# 汽车租赁管理系统  
PRG2201 面向对象编程 — Coursework #3  
INTI 国际大学 • 2025 年 8 月学期

## 📌 项目简介
本项目要求开发一个 **基于 Java GUI 的汽车租赁管理系统**。  
系统包含车辆库存管理、租借流程、用户管理、数据持久化和报表功能。此项目占 **课程评分的 30%**。

## 👥 用户角色与功能

### 1. **管理员（Admin）**
- **用户管理：** 添加、修改、删除与查看员工账号  
- **车辆管理：** 新增、修改与删除车辆  
  - 信息包含：品牌、型号、年份、车牌、颜色、状态  
- **生成报表：**  
  - 租借记录  
  - 车辆库存状态  
  - 收入统计  
- **系统日志：** 查看系统操作记录并排查问题

### 2. **员工（Staff Member）**
- **租借管理：**  
  - 办理车辆租出  
  - 办理车辆归还  
  - 记录租借起止日期  
- **车辆状态：** 查看与更新车辆可用情况  
- **搜索功能：** 按品牌 / 型号 / 车牌搜索车辆  
- **顾客管理：** 查看与修改顾客信息及租借历史

## 🗄️ 数据管理
- 必须使用 **关系型数据库**（MySQL / PostgreSQL / SQLite）。  
- 存储以下数据：  
  - 车辆信息  
  - 用户账户  
  - 租借记录  

### 若无法连接数据库需提供理由
例如：技术限制 / 没有服务器资源 / 配置时间不足。  
可使用文件系统（CSV、序列化等）代替，但需在报告中说明限制。
